# Proyecto Triangulos

## Descripción
Programa en Java que identifica tipos de triángulos y calcula áreas.

## Lenguaje
Java

## Funciones
- tipoTriangulo: identifica el tipo de triángulo
- areaTriangulo: calcula el área del triángulo

## Como ejecutar
javac Triangulo.java
java Triangulo
